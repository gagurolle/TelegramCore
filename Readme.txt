Бот, отправляющий курс BTC_USD

в файле config.ini хранится id чата администратора

в файле chat_list_id.txt хранятся id чатов, которые подписаны на бота.

Также, прежде чем запуска   1ть бота, следует создать папку C:\ChromeDriver\ и положить в нее актуальный chromedriver.exe

Сейчас бот крутится на AWS windows server 2016, файлы передаются посредством соединения ssh. Идет работа над созданием скриптов по автоматическому запуску при изменении программы. 

Также проект делается на .net core, потому что в последствии будет перенесен на платформу arm64
